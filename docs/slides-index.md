# スライド一覧（受講後の復習用）

受けた回のスライドを、あとから見返せます。各スライドは**投影版と同じもの**で、リンクは**別タブで開きます**（矢印キーでページ送り、`F`でフルスクリーン、スクロールしません）。

!!! tip "使い方"
    受講した回のスライドを開いて、**プロンプトや手順を復習**しましょう。まだ受けていない回の**予習**にも使えます。各回の狙い・卒業条件は [カリキュラム設計書](design/curriculum.md) を参照。
