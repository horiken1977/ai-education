# セッションの学び（復習）

!!! note "このページの方針"
    - ここに載せるのは **議論のテーマと学び**です。**「誰が何を言ったか」は書きません**（匿名）。
    - 録画そのものや発言録は公開しません。**学びの構造だけ**を残します。
    - 具体的な社名・数値・個人が特定できる話は含めません。
