# オンボーディング台本（全員必須・L1の前に1回）
### テーマ：「AIは怖くない。一緒に“身につけて”いこう」
**今日のゴール宣言：終わったら、①AIを自分の手で動かして「これは使えそう」と感じ、②この研修への自分なりの向き合い方（自分ごと）を1つ言葉にできる**

> 共通ルール（安全・進め方・記号の意味）は [台本ガイド](index.md) を参照。

> **この回の設計思想（講師向け・最重要）**：これは“知識を教える”回ではなく、**マインドセットを「体験を通して」渡す**回。
> 冒頭に目的・ゴールを10分説明しただけでは納得は得られない。**先に手を動かして「なるほど、これは使えそう」を体感**させ、「AIってよく分からない・怖い・信じられない」という最初の壁を超えることが本当のゴール。
> 伝えたいメッセージは“言う”のではなく、**やってもらった直後に、その体験に紐づけて短く言葉にする**。講師が話す合計は15分以内。L1の内容と重複して構わない。

### この回で必ず届ける6つのメッセージ
1. AIを使えることはこれからの必須スキル。**学び続け・変わり続ける**行動を取ろう。
2. ゴールは「わかる」でも「できる」でもなく、その先の **「身につく」**。
3. これは私が教える場ではない。**私も一緒に学び・身につけ・上達していく**。受講者どうしの**学び合い**が何より大事。
4. レベルは段階構成（LV1〜LV3＋エクストラ）。**毎回どのレベルに出るかは自分で選ぶ**（前半LV1多め→中盤LV2→後半LV3、行き来も自由）。
5. **カリキュラムは途中で変わる**。AIの進化が桁違いに速いから（変化に乗れること自体がスキル）。
6. （上記すべてを）**手を動かして腑に落とす**。説明より体験。

> ※この回で **研修の地図（レベル定義・業務フローの背骨・3ヶ月の流れ）** も共有する（受講生が毎回自分でレベルを選べるように）。詳細は [カリキュラム設計書](../design/curriculum.md)。

### わかる → できる → 身につく の階段（この研修の背骨）
| 段階 | 状態 | どこで効くか |
|---|---|---|
| **わかる** | AIで何ができるか腑に落ちる | オンボーディング前半 |
| **できる** | 手伝ってもらえば一度できた | オンボーディング後半・L1 |
| **身につく** | 考えなくても普段使いになる | **毎週の反復・宿題（ここが本体）** |

### 事前準備チェック（開始前にTAが確認）
- [ ] 全員 Copilot にログイン済み（前日Forms提出を確認）
- [ ] 配布物：`オンボーディング_サンプル文.txt`（要約用の長文＋言い換え用のぶっきらぼう文＋L1ライト用の架空Teams会議トランスクリプト）
- [ ] レベル診断 Forms（5問）のリンクを用意（後半でその場回答）
- [ ] TAは開始10分前から待機ルームでログイン未完了者をフォロー

### 0:00–0:05　ウェルカム＆「今日は違う」宣言
- 🎤「ようこそ。最初に一番大事なことを言います。**今日も、この研修全体も、私が一方的に“教える”場ではありません。私も皆さんと一緒に学んで、身につけて、上達していく仲間です。**だから私が知らないことも普通に出てきます。それを一緒に面白がりましょう。」
- 🎤「もう1つ。私一人が教えるより、**この20人がお互いに教え合う**方が、何倍も速く上手くなります。今日からみんながお互いの先生です。」
- 🎤（安全ルール30秒を読む）
- 🎤「今日のゴールは2つだけ。①AIを自分の手で触って『お、使えそう』と感じること。②研修との自分の向き合い方を、自分の言葉で1つ決めること。説明は最小限、すぐ触ります。」
- 💬チャット：`今日のゴール：①AIを触って「使えそう」を体感 ②研修との向き合い方を1つ決める。準備：Copilotを開いて✅`

### 0:05–0:12　ファーストコンタクト（説明より先に、まず触る）
- 🎤「理屈は後。まず1個、AIに頼んでみましょう。下のメニューから1つ選んで打つだけ。正解はありません。何を打っても壊れません。」
- 💬チャット（3択メニュー・コピペ可）：
  ```
  【どれか1つ選んで打ってみよう】
  ① 要約：（配布の長文を貼って）「これを3行で要約して。日本語で。」
  ② 説明：「"○○"（前から曖昧だった言葉）を、中学生にも分かるように説明して。」
  ③ 言い換え：（配布のぶっきらぼうな一文を貼って）「これを丁寧で感じのいい文に直して。」
  ```
- 🎤「返ってきたら、続けて一言『もっと短く』とか『例えを入れて』と返してみてください。…**ね、会話できるでしょう?** 命令じゃなくて、相談する感じです。」
- 🧑‍🔧TA：✅監視。❓には即対応。誰も置いていかない。
- ⚠つまずき：「怖くて何を打てばいいか分からない」→ TA「メニュー①をそのままコピペでOKです」。

### 0:12–0:20　体験の振り返り → マインドセットを“体験から”渡す
- 🎤「今、何が起きましたか? 皆さん、AIに『命令』というより『お願い・相談』してましたよね。それでいいんです。」
- 🎤（メッセージ①）「実は今やったこと、1〜2年前のAIではうまくできませんでした。**AIは毎月変わります。だから私たちも“学び続け・変わり続ける”前提でいきましょう。**」
- 🎤（メッセージ②）「この研修のゴールは、『わかる』でも『できる』でもなく、その先の **『身につく』**。歯磨きみたいに、考えなくても普段使いになる状態が目標です。」
- 🖱操作／💬チャット：階段の図を提示。
  ```
  わかる   → できる   → 身につく
  (何ができるか)(一度できた)(普段使いになる＝ゴール)
   今日前半     今日後半・L1    毎週の反復・宿題でここへ
  ```
- 🎤（メッセージ③）「そして、ここへ行くのに一番効くのが“学び合い”。さっきの自分の結果、隣の人のと見比べたくなりませんか? それを今日この後やります。」

### 0:20–0:35　ブレイクアウト①：ペアで“これは使えそう”を体験（L1ライト）
- 🎤「ここから、もう少し仕事に効くやつを。大事な前提を1つ——**会議はメモを取らず、Teamsで録画してCopilotに文字起こしを作らせる**のが基本です。今日はその文字起こしを用意しました。これをAIに『議事録・上司報告・次アクション』へ整えてもらいます。ペアで『これ自分でやったら何分かかる?』を話してください。15分。」
- 🎤（録画の同意・必ず添える）「※お客様との打ち合わせを録画するときは、**録画の前に必ず**『記録のために録画させてください。私たちの記録のためだけに使います』と断って、**合意を得てから**にしてください。」
- 💬チャット（コピペ可）：
  ```
  【ペアワーク】15分
  1. 配布の会議トランスクリプトを貼って、次を打つ：
     「あなたは法人営業の事務サポートです。このTeams会議の文字起こしを
      ①議事録 ②上司への3行報告 ③次アクション一覧（やること/担当/期日）
      に整えて。無いことは“要確認”、事実は作らないで。」
  2. 結果を相手に画面共有
  3. 「メモを取らずにここまで出せた」「どこが便利?」を話す
  ```
- 🧑‍🔧TA：各ルームを巡回。詰まりを1分救済。良い反応・良い例を2つピック（後の共有用）。
- ⚠つまずき：「メモが整いきらない」→ TA「1行でも大丈夫。AIが“要確認”で補います」。

### 0:35–0:43　全体共有＋「私も学んでいる」を体現
- 🎤「2名、見せてもらえますか。」（TAがピックした2名）→ 反応を拾う。
- 🎤（メッセージ③を体現＝弱さを見せる）「正直に言うと、私もこの使い方を知ったのは最近です。きっと皆さんの中に、私より上手い人がいます。**今日みたいに見せ合うのが、一番速い学び方**なんです。」
- 🎤（自分ごとへの橋）「想像してください。これが毎日の議事録・報告で効いたら、1日に何分戻ってきますか? その時間で、もう1件商談できますよね。**“使えそう”を“自分の得”に変換**してみてください。」

### 0:43–0:50　研修の地図：レベル定義・業務フロー・3ヶ月の流れ
- 🎤「研修の全体像です。営業の仕事を“どの流れでAIに渡すか”で組んでいます。」
- 🖱/💬：業務フロー×AIの図を見せる → `①調べる → ②記録する → ③書く → ④見せる → ⑤回す → ⑥作る`。**今日やった②記録がスタート地点**。
- 🎤「レベルはこの流れに沿って上がります（地図なので駆け足で）。」
  - **オンボーディング（今日）**＝②記録する＝AIに“素材”を渡す
  - **LV1**＝チャットAI（ICA/Copilot）を使い倒す＝①調べる・③書く
  - **LV2**＝Bobでデモを作りお客様に見せる＝④見せる
  - **LV3**＝ICA/Copilot/Bobを使い分けて案件を回す＝⑤
  - **エクストラ（希望者）**＝Bobで簡単なAIアプリ＝⑥作る側
- 🎤「渡し方の“型”は1つだけ：**素材を渡す → 指示 → AIに作らせる → 自分で確認・仕上げ**。レベルが上がっても、渡す仕事と使うAIが広がるだけです。」
- 🎤（メッセージ④）「3ヶ月の進み方は、**毎回みなさんが自分でレベルを選ぶ**形。前半はLV1多め、中盤はLV2、後半はLV3が増えます。各レベルは何度もあるので、**行ったり戻ったり、自分のペースでOK**。」
- 🖱操作：レベル診断（5問）をその場で30秒。「次に出る回を選びましょう。」
- 🎤（メッセージ⑤）「もう1つ約束を。**このカリキュラムは途中で変わります。** AIの進化が速すぎるから。変わったら『お、来たな』と。**変化に乗れること自体がスキル**です。」
- 💬チャット：`レベル診断Forms（30秒）→ [リンク] ／ 早見：LV1=チャットAI使い倒し｜LV2=Bobでデモ｜LV3=使い分け｜EX=AIアプリ ／ 3ヶ月＝前半LV1・中盤LV2・後半LV3（毎回自分で選ぶ）`
