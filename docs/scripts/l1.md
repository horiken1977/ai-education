# L1：初心者セッション台本
### テーマ：「AIに頼んで、仕事の一部を終わらせる」
**今日のゴール宣言：終わったら、商談後の殴り書きメモを、整った議事録・上司報告・次アクション一覧に“自力で”整形できる**

> 共通ルール（安全・進め方・記号の意味）は [台本ガイド](index.md) を参照。配布教材は [L1 商談メモ](../materials/l1-shodan-memo.md)。

> **設計メモ（講師向け）**：L1ではあえて「AIにメール返信を書かせる」等の“生成”タスクを使わない。背景を知らないAIの生成物は当たり障りがなく修正に時間がかかり、初体験を台無しにするため。
> ここでは **参加者が中身（商談メモ）を持ち込み、AIは整形・構造化に徹する**。だから初心者でも確実に時短を体感でき、ハルシネーションも起きない。「自分のメモが3分で出せる議事録になった」が今日の感動ポイント。

### 事前準備チェック（開始前にTAが確認）
- [ ] 全員 Copilot にログイン済み（前日Forms提出を確認）
- [ ] 配布物：`L1_サンプル商談メモ.txt`（架空・殴り書き風）をTeamsチャネル/チャットに添付済み
- [ ] 参加者へ事前案内：「直近の自分の商談メモ（無ければ社内MTGのメモでも可）を1つ手元に。顧客名は伏字で」
- [ ] OneNote「個人プロンプト帳」セクションのリンクを用意
- [ ] TAは開始10分前から待機ルームでログイン未完了者をフォロー

### 0:00–0:05　チェックイン＆ゴール宣言
- 🎤「こんにちは。今日のゴールは1つだけ。**終わるころには、商談後の殴り書きメモが、そのまま出せる議事録・上司報告・次アクション一覧に変わります**。難しい知識はいりません。みなさんが持っているメモを、AIに整えてもらうだけです。」
- 🎤「ポイントは、**中身を考えるのはみなさん、整えるのがAI**。だから的外れになりません。」
- 🎤（安全ルール30秒を読む）
- 💬チャット：`今日のゴール：商談メモ→議事録・報告・次アクションに整形。準備：Copilotを開く＋自分の商談メモを手元に📝`
- 🧑‍🔧TA：チャットで「Copilot開けた人は✅を押してください」と促し、反応を数える。

### 0:05–0:12　ミニデモ（講師は1個だけ実演）
- 🖱操作：Copilotを開き、配布の`L1_サンプル商談メモ`（箇条書きの殴り書き）を貼り付け、次を実演。
     ```
     あなたは法人営業の事務サポートです。
     次は私が商談直後に書いた殴り書きメモです。これを3つの形に整えてください。
     ① 議事録（日時・出席者・決定事項・論点が分かるように）
     ② 上司への報告（3〜4行、結論から）
     ③ 次アクション一覧（やること／担当／期日 の表）
     不明な項目は「要確認」と書いてください。事実を勝手に作らないこと。
     ```
- 🎤「見てください。**ぐちゃぐちゃのメモが、3分でこの3点になりました**。中身は私のメモのまま、形だけAIが整えています。最後の『事実を作らないで』が大事で、知らないことは“要確認”と書かせます。」
- 💬チャット：`▼今コピーして使うプロンプト（メモ整形3点セット）`（上記を貼る）

### 0:12–0:22　写経ワーク（全員が同じ題材で成功体験）
- 🎤「では同じサンプルメモで、みなさんも3点セットへの整形をやってみましょう。10分取ります。できたらチャットに✅。詰まったら❓を押してください、TAが拾います。」
- 🧑‍🔧TA：チャットを監視。❓には即DM/返信。✅の数で進捗把握。3分経って✅が半数未満なら講師に合図。
- ⚠つまずき：「英語で返ってくる」→ プロンプト末尾に`日本語で。`を足す、とチャットで案内。
- ⚠つまずき：「項目が埋まらない/勝手に埋めてる」→「メモに無いことは“要確認”でいい、と指示に入っていますか？」と確認を促す。
- 🎤（残り2分）「できた人は、次アクションの表に**“優先度”列を足して**と打ってみてください。表が育ちます。」

### 0:22–0:37　ブレイクアウト①：ペアワーク（自分ごと化）
- 🎤「ここからペアに分かれます。**自分の直近の商談メモ（顧客名は伏字に）**を同じプロンプトで整形して、お互いの画面を見せ合ってください。15分です。」
- 💬チャット（BO内に表示）：
  ```
  【ペアワーク手順】15分
  1. 自分の商談メモを1つ用意（社名・氏名は○○に伏字／無ければ社内MTGメモでOK）
  2. 3点セット（議事録／上司報告／次アクション）に整形
  3. 相手に画面共有して「自分でやるより何分くらい速くなった?」を伝え合う
  ```
- 🧑‍🔧TA：各ルームを巡回。詰まっているペアに入って1分救済。良い事例をメモ（後で全体共有用に2例ピック）。
- ⚠つまずき：「メモが断片的で整わない」→ TA「箇条書き1行でも大丈夫。AIが“要確認”で補ってくれます」。
- ⚠つまずき：「伏せる情報が分からない」→ TA「社名・氏名・金額・固有の製品名を○○に。文脈は残してOK」。

### 0:37–0:46　全体共有＋プロンプトの「型」
- 🎤「2名にお願いします。画面を見せてもらえますか。」（TAがピックした2名）
- 🖱操作：共有を見ながら、ホワイトボード/チャットに**プロンプトの型**を提示。
  ```
  ①役割（あなたは〜の事務サポートです）
  ②素材（これは私が書いたメモです＝中身は自分が持ち込む）
  ③依頼（何にしてほしい：議事録／報告／表）
  ④出力形式（表・3行・箇条書き など）
  ＋ 安全弁：「無いことは“要確認”、事実を作らない」
  ```
- 🎤「中身を持っているのはみなさん。AIは**形を変える係**です。だから的外れになりません。今日の魔法はこれだけです。」
