# LV1：チャットAI使い倒し 台本
### テーマ：「チャットAI（ICA / Copilot）を、毎日の相棒にする」
**ゴール：営業の日常業務（調べる・書く・壁打ち）をチャットAIに渡して回せるようになる**

> 共通ルール（安全・録画同意・進め方・記号）は [台本ガイド](index.md) を参照。レベルの位置づけは [カリキュラム設計書](../design/curriculum.md)。
> 配布教材：[LV1 リサーチ対象サンプル](../materials/lv1-research-target.md)

> **LV1は3回ぶんの“引き出し”を持つ**（毎回テーマ＝営業業務を変えるので、複数回出ても重複しない）：
> - **LV1-a 調べる**（商談前リサーチ）… この台本のメイン（分刻み）
> - **LV1-b 書く**（メール・要約・提案下書き）… §後半に進行アウトライン
> - **LV1-c 使い倒す**（壁打ち・ロープレ・ICA/Copilot使い分け）… §後半に進行アウトライン
>
> **渡し方の型（全レベル共通・LV1でも毎回確認）**：素材を渡す → 指示（役割・依頼・出力形式・安全弁）→ AIに作らせる → 自分で確認・仕上げ。

## LV1-a：調べる（商談前リサーチをAIに渡す）— 60分・分刻み

- **今日のゴール宣言**：「終わったら、商談前リサーチ（企業・業界・担当者・課題仮説）をチャットAIに渡して、**10分で準備**できる」
- **使うツール**：ICA（深い調査・提案の下ごしらえ）／ Copilot（手早く）。両方触って違いを体感。

### セリフ・操作の要点
- 🎤（0:05 デモ前）「リサーチはAIが一番速くなる作業の1つ。ただし**鵜呑みは厳禁**。だから最後に『確認すべき点を挙げて』を必ず付けます。」
- 🖱操作（デモ）：ICAに配布の架空企業概要を貼って、次を実行。
  ```
  あなたは法人営業のリサーチ担当です。
  次の企業について、商談前に押さえるべき点を整理してください。
  ① 業界の動向と、この企業が直面しそうな課題（仮説）
  ② キーパーソン仮説（役割・関心事）
  ③ 当社が刺さりそうな切り口（仮説）
  ④ 私が事前に裏取りすべき「確認すべき点」
  推測の箇所は「推測」、不明は「要確認」と明記。事実を断定しないこと。
  ```
- 🎤「同じ依頼をCopilotにも投げて、**速さと深さの違い**を見ます。じっくりはICA、手早くはCopilot。」
- 🧑‍🔧TA：BO①巡回。「裏取り対象（数字・固有名詞・最新動向）」を1つは挙げさせる。
- ⚠つまずき：「それっぽい嘘が混じる」→「だからリサーチは“仮説の下書き”。確認すべき点を出させて、人が裏取りする前提でOK」。
- 💬チャット（宿題）：
  ```
  【宿題：次回まで】
  次の商談の前に、必ずAIでリサーチ（①〜④）してから臨む。
  記録：役立った点／AIが外した点（裏取りで判明）／時短できた分
  ```

- **持ち帰り成果物**：自分の担当先1社の「リサーチ＋確認すべき点」メモ
- **卒業（LV1全体）に近づく目安**：リサーチを毎回の商談前ルーティンにできる

## LV1-b：書く（メール・要約・提案下書きをAIに渡す）— 進行アウトライン
- **ゴール**：日常の文章作成（返信メール・長文要約・提案の骨子下書き・FAQ）をAIで回す。
- **型**：素材＝自分の意図・箇条書き・元文を渡す → AIが整える（中身は人、形がAI）。
- 0:05 デモ：箇条書きメモ→提案の骨子下書き／長文メール→要点＋返信案。
- 0:22 BO①：自分の実務文章（社外秘は伏字）で1本。
- 0:46 応用：トーン違い・長さ違い・宛先違いで“言い直して育てる”。
- 宿題：実務の文章3本をAIで下書き→自分で仕上げて送る。

## LV1-c：使い倒す（壁打ち・ロープレ・使い分け）— 進行アウトライン
- **ゴール**：AIを「相棒」にする。アイデア出し・商談ロープレ・反論対応の壁打ち＋ICA/Copilotの使い分けを自分の言葉で言える。
- 0:05 デモ：AIを顧客役にしたロープレ（「予算に厳しい購買担当として反論して」）→フィードバックをもらう。
- 0:22 BO①：自分の提案で壁打ち。相手ペアと「AIに何を相談すると効くか」を共有。
- 0:43 使い分けの言語化：日常・手早く＝Copilot／深い調査・提案＝ICA。各自1文で書く。
- 宿題：1週間、毎日1回はチャットAIに業務を渡す（調べる/書く/壁打ちのどれか）。

### LV1 共通：宿題と卒業条件
- **宿題（毎週）**：日常業務を毎日1回はチャットAIに渡す。記録（使った業務／効いた点／詰まった点）。
- **卒業条件**：リサーチ・要約・メール・下書き・壁打ちを**自分のプロンプトで週次常用**している（→ 診断で LV2 へ）。
