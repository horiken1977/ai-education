> 台本：[オンボーディング](onboarding.md) ／ [LV1 チャットAI使い倒し](l1.md) ／ [LV2 Bobでデモ](l2.md) ／ [LV3 使い分け](l3.md) ／ [エクストラ Bobでアプリ](extra.md)

## この台本の使い方
- **「🎤セリフ」** はそのまま読める話し言葉。固すぎたら自分の言葉に崩してOK。
- **「🖱操作」** は講師の画面で見せる具体操作。**「💬チャット」** はTAまたは講師がチャットに貼る文。
- **「🧑‍🔧TA」** は共同ホストの動き。**「⚠つまずき」** は想定トラブルと即応。
- 講師が話す時間は**合計15分以内**。それ以外は「参加者が触っている」状態を守る。
- 基盤はM365：診断＝**Forms**、テンプレ保存＝**OneNote個人セクション**、共有＝**Teamsチャネル/SharePoint**、集計＝**Excel/Power BI**。

### 全レベル共通・冒頭30秒の安全ルール（毎回読む）
> 🎤「最初に1つだけお約束です。**お客様名・個人情報・社外秘は、そのまま入力しない**でください。今日は伏字か、配布した架空データを使います。判断に迷ったらチャットでTAに聞いてください。」

### 録画とトランスクリプトの基本ルール（会議を扱う回で必ず伝える）
- 会議メモは取らなくてよい。**会議はTeamsで録画 → Copilotでトランスクリプト／会議まとめ** を作り、それをAIの素材にする（この会社はメモを取る習慣がないため、ここを徹底して推す）。
- **お客様との打ち合わせは、録画の前に必ず同意を得る**。言い方の例 →「**記録のために録画させてください。私たちの記録のためだけに使います。**」合意を得てから録画する。
- 社内会議も録画を基本にしてOK。長いトランスクリプトの要約は取り違いが起きうるので、重要論点は録画と突き合わせて確認する（プロンプトに「無いことは“要確認”、事実を作らない」を必ず入れる）。

## 付録A：Microsoft 365 基盤セットアップ（運営者向け）

受講者が使う**共有プロンプトライブラリ**と**進捗ダッシュボード**をM365で完結させる構成。追加コストなしで運用できます。

### A-1. 共有プロンプトライブラリ
| 用途 | 置き場所 | 補足 |
|---|---|---|
| 個人のプロンプト帳 | **OneNote**（各自のセクション） | L1から各自が貯める |
| チームの共有テンプレ | **Teams「AI共有テンプレ」チャネル** | L2で投稿。検索しやすい |
| 業務資産（手順書付き） | **SharePoint「営業AI資産ライブラリ」** | L3の成果。ドキュメントライブラリ＋メタデータ（業務種別・作成者）で整理 |

### A-2. レベル診断 → ダッシュボードの配線
1. **Microsoft Forms** で5問のレベル診断を作成（[カリキュラム設計書](../design/curriculum.md)参照）。毎セッション募集時に配布。
2. Forms回答は **Excel に自動連携**（Forms内「Excelで開く」）。
3. Excelで **レベル判定列**（IF式で L1/L2/L3 を自動付与）と **週次の集計表** を作る。
4. **Power BI（または Excelのグラフ）** で **レベル分布の推移折れ線** を1枚に。
   - GitHubで管理する場合は、集計を CSV にして [KPIダッシュボード](../dashboard/index.md) に取り込む（[GitHub運用設計](../design/github-ops.md)参照）。

### A-3. レベル自動判定のExcel式（例）
診断Q1〜Q5を1（はい）/0（いいえ）で集計し、合計点で判定する簡易版：
```
=IF(合計<=1,"L1", IF(合計<=3,"L2","L3"))
```
※運用しながらしきい値は調整。卒業条件（設計書）と整合させる。

### A-4. 毎週の運営ルーティン
- 月：診断Forms配布／前回宿題リマインド（Teams）
- 火〜水：未セットアップ者をTAがフォロー
- 木 or 金：本番セッション
- 翌日：ダッシュボード更新／良かった資産をライブラリに昇格／次週レベル配分を決定

## 付録B：講師の心構え（毎回これだけ）
1. **話しすぎない**（合計15分以内）。沈黙を恐れず、参加者が触る時間を守る。
2. **完璧な出力を見せない**。直していく姿を見せるほうが「やれる感」が出る。
3. **✅と❓で全員の状態を可視化**。置いてけぼりを作らない。
4. **毎回1つの成果物を必ず持ち帰らせる**。手ぶらで帰さない。
5. **宿題が本体**。「使った人が上手くなる」を毎回言い切る。
