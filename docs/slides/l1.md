# LV1 チャットAI使い倒し
## 定義：チャットAIでディスカッション・ペーパー（PPT）を作れる

**このレベルが終わったら…**
提案・商談用のディスカッション・ペーパーを「調べる → 構成 → スライド化 → 仕上げ」で
**AIに渡して1本作れる**（社内・顧客情報は伏字してGensparkへ）

<small>営業フロー上の担当：①調べる・③書く（マテリアル準備）</small>

---

## LV1の実施の流れ（全3回・各60分）

| 回 | テーマ | この回でやること |
|---|---|---|
| **a** 中身をつくる | 調べる＋構成 | リサーチ→提案の構成案をAIに渡して用意（ICA / Copilot） |
| **b** スライドにする | Copilotで伏字 → Genspark | 構成案からPPTを1本生成（伏字・一般化してGensparkへ） |
| **c** 自案件で仕上げる | 一気通貫＋磨き | 保存したプロンプトを再利用し、担当先のディスカッション・ペーパーを通しで作る |

> 調べる・書くは**PPTの前工程**。1本のワークフローに統合します。
> **どの回からでもOK＝各回1回完結**。b/c は前工程の成果物（開始素材）を配布。連続で出た人は自分の成果を継ぎ足せます。

---

## グラウンドルール（毎回、最初に確認）

**この場の“進め方の約束”。毎回ここから始めます。**

- 🏃 **考えるより、まず行動** — 手を動かしながら考える
- ✏️ AIへの指示は**最初はグダグダでOK**。やりながら直す
- ⚡ **スピード最優先**
- ⭕ **間違いはない。全部“正解にする”**
- 🤝 講師も一緒に学ぶ**仲間**（ただの案内人）
- 💬 **対話で深める**（ブレイクアウトを多用します）
