# 営業向けAI研修プログラム

営業職向けに、**IBM Consulting Advantage** と **Microsoft Copilot** を使いこなすためのハンズオン研修プログラムです。
1セッション60分・週1回・3ヶ月で、**「初心者ゼロ・受講者の半数以上が上級」** へ引き上げることをゴールにしています。

## このサイトの使い方
- **講師・運営の方** → まず [カリキュラム設計書](design/curriculum.md)、当日は [講師台本](scripts/index.md)。
- **受講者の方** → 自分のレベルの[配布教材](materials/l1-transcript.md)と[テンプレート集](templates/index.md)を使ってください。
- **進捗を見たい方** → [KPIダッシュボード](dashboard/index.md)。

## まず全員：オンボーディング
レベル別に入る前に、**全員が「オンボーディング」を1回受講**します。AIを自分の手で触って「これは使えそう」を体感し、**「学び続ける／“身につく”をゴールにする／一緒に学び合う」** マインドセットを持つ回です。→ [オンボーディング台本](scripts/onboarding.md)

## 3つのレベル（どこから受けてもOK）
| レベル | 一言ゴール | 台本 |
|---|---|---|
| **L1 初心者** | 「AIに頼んで仕事の一部が終わった」を体験する | [L1台本](scripts/l1.md) |
| **L2 初級〜中級** | 自分の営業業務に AI を組み込む型（テンプレ）を持つ | [L2台本](scripts/l2.md) |
| **L3 上級** | AIを仕組み化・横展開して時間と質を稼ぐ | [L3台本](scripts/l3.md) |

## 進め方の原則
- 講師のプレゼンではなく **ハンズオン**。毎回 **1つの成果物**を必ず持ち帰る。
- 上達のエンジンは授業ではなく **宿題（実務での実使用）と横展開**。
- 「使う → 型化 → 教える」の上昇スパイラルで、3ヶ月かけてレベルを引き上げる。
