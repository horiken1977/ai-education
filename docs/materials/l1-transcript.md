# L1 配布教材：Teams会議トランスクリプト サンプル

L1セッションの写経ワークで使う、**架空のTeams会議録画トランスクリプト（文字起こし）**です。参加者はこれを Copilot に貼り付けて、議事録・上司報告・次アクション一覧に整形します。

!!! tip "本番の進め方（受講者へ）"
    会議でメモは取らない。**会議をTeamsで録画 → Copilotでトランスクリプト／会議まとめを生成** → それをAIの素材にする。
    お客様との打ち合わせは、録画の前に必ず同意を得る →「**記録のために録画させてください。私たちの記録のためだけに使います。**」合意を得てから録画。

!!! note "ステータス：サンプル未作成（次の一手で作成）"
    ここに架空のTeams会議トランスクリプト本文を作成して配置します。要件：

    - 完全に**架空**（実顧客名・実案件・金額を使わない）
    - **Teamsの自動文字起こし風**（話者ラベル＋発言が続く形式。相づち・言い直し・脱線を少し残し“生”っぽく）
    - 日時・出席者・先方の課題・決定事項・宿題・次回予定が拾える情報量
    - 整形すると「議事録／上司報告／次アクション表」の3点が作れる素材を含む

（配布用は `.txt` でも同内容を用意し、Teams チャネルに添付）
