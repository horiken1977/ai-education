# L1 配布教材：商談メモサンプル

L1セッションの写経ワークで使う、**架空の殴り書き商談メモ**です。参加者はこれを Copilot に貼り付けて、議事録・上司報告・次アクション一覧に整形します。

!!! note "ステータス：サンプル未作成（次の一手で作成）"
    ここに架空の商談メモ本文を作成して配置します。要件：
    
    - 完全に**架空**（実顧客名・実案件・金額を使わない）
    - わざと**断片的・箇条書きの殴り書き**にする（整形のビフォーが分かるように）
    - 日時・出席者・先方の課題・決定事項・宿題・次回予定が拾える情報量
    - 整形すると「議事録／上司報告／次アクション表」の3点が作れる素材を含む

（配布用は `.txt` でも同内容を用意し、Teams チャネルに添付）
