# L2 配布教材：案件リスト

L2セッションの Copilot in Excel パートで使う、**架空の案件リスト**です。並べ替え・優先度付け・集計を体験します。

!!! note "ステータス：サンプル未作成（次の一手で作成）"
    要件：
    
    - 完全に**架空**の案件・社名
    - 15〜30行程度。列の例：案件名／顧客（架空）／金額／受注確度／次アクション／期日／担当
    - 「受注確度が高い順に並べ替え」「今週優先3件」などの指示が映える内容
    - 配布は **`L2_案件リスト.xlsx`**（Excelでテーブル化しておくと Copilot が効きやすい）

（この Markdown は説明用。実データは `.xlsx` で配布）
