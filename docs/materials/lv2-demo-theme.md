# LV2 配布教材：デモ業務テーマ（架空）

LV2-a（Bob入門＋初めての1本）の写経ワークで使う、**架空の業務シナリオ**です。参加者はこれを Bob に渡して、「◯◯業務のデモを作って」と話しかけ、紙芝居レベルの画面プロトを作ります。

!!! tip "本番（受講者へ）"
    お客様の**実データ・社外秘は入れない**。架空データ・架空業務で作る。
    完璧なアプリは不要。**紙芝居レベル**で「こんな感じ」を体感させるのが目的。見せ方は画面提示でOK。

!!! note "ステータス：サンプル未作成（次の一手で作成）"
    ここに架空の業務シナリオを2〜3本作って配置します。各シナリオの要件：

    - 完全に**架空**（実社名・実データなし）
    - 「誰が・何の画面で・何ができる」が1〜2行で分かる粒度
    - 営業が出会いがちな業務（例：在庫管理／問い合わせ受付／案件進捗ボード 等）を想定
    - そのまま `「◯◯業務のデモを作って。一覧と登録ボタンがある感じで」` の1行に落とせること

    例（雛形）：
    ```
    テーマ名：架空・在庫管理
    使う人：倉庫担当者
    画面：在庫一覧／登録フォーム
    やりたいこと：在庫を見る・新しい入荷を登録する
    → Bobへの一言：「在庫管理業務の、担当者が使うデモ画面を作って。一覧と登録ボタンがある感じで」
    ```
