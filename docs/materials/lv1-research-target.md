# LV1 配布教材：リサーチ対象サンプル（架空企業）

LV1-a（調べる）の写経ワークで使う、**架空企業の概要メモ**です。参加者はこれを ICA / Copilot に渡して、業界動向・課題仮説・キーパーソン仮説・刺さる切り口・確認すべき点を出させます。

!!! note "ステータス：サンプル未作成（次の一手で作成）"
    ここに架空企業の概要メモを作成して配置します。要件：

    - 完全に**架空**（実社名・実数値は使わない）
    - リサーチの“素材”になる最小限の前提：業種・規模感・事業内容・最近の動き（架空）・商談相手の部署など
    - これ1枚から「①業界動向と課題仮説 ②キーパーソン仮説 ③刺さる切り口 ④確認すべき点」が生成できる情報量
    - わざと情報を“やや不足”にして、AIに「要確認」を書かせる余地を残す

!!! tip "本番（受講者へ）"
    リサーチは**公開情報のみ**。社外秘・未公開情報は入力しない。AIの出力は“仮説の下書き”として、数字・固有名詞・最新動向は必ず人が裏取りする。

（配布用は `.txt` でも同内容を用意し、Teams チャネルに添付）
